Privacy.

Palimpsest is a high-performance PDF utility built with a strict "client-first" philosophy.

100% Local

Traditional tools upload sensitive documents to a remote server. Palimpsest never does. Files are processed entirely within the browser's local memory (RAM). No data ever leaves the device.

Technical Core

Palimpsest leverages industry-standard libraries like PDF.js for high-fidelity rendering and pdf-lib for surgical modifications to the document structure. High-speed operations are offloaded to Web Workers to ensure a smooth, lag-free experience.

Minimalist & Open

Designed for efficiency and transparency. No tracking, no cookies, no accounts. Just function. The source code is open and available for audit.
